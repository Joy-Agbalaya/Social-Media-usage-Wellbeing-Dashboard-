# Social-Media-usage-Wellbeing-Dashboard

I analysed the relationship between social media usage, sleep, stress, and productivity, showing how these factors connect rather than exist in isolation
 
 Problem:
Understand how social media usage impacts sleep, stress levels, and daily productivity.

 Data Source:
Survey data collected from 11,000 respondents (Kaggle datasets)
 Tools Used
- Excel (for cleaning and visualisation)

 Key Insights:
1. Sleep Impact: Users with >4hrs daily social media averaged 1.5hrs less sleep  
2. Stress Correlation: High Instagram/TikTok usage correlated with 23% higher reported stress  
3. Productivity: Peak productivity hours dropped when the social media use increased  

## Main Takeaway
The data does not show social media is purely "bad" or "good". Instead, the healthiest outcomes for sleep, stress, and productivity were linked to **moderate usage**. Both very high usage and complete avoidance showed negative patterns. Balance, not extremes, drives wellbeing.
